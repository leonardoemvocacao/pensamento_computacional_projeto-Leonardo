# Sistema de Açaiteria

Este projeto foi desenvolvido em Python com o objetivo de simular o funcionamento de uma açaiteria, permitindo a interação do cliente com o sistema através do terminal.

## Funcionalidades

* Cadastro de Clientes - Feito por Wilson
* Visualização do Cardápio - Feito por Gustavo
* Realização de Pedidos - Feito por Rhuan
* Chat com Atendimento - Feito por Leonardo
* Área de Ajuda - Feito por Guilherme Firmino

## Tecnologias Utilizadas

* Python 3

## Como Executar

1. Baixe o arquivo `projeto_app_acaiteria.py`
2. Abra o terminal na pasta do projeto
3. Execute o comando:

```bash
python projeto_app_acaiteria.py
```

## Como Funciona

O sistema possui um menu principal onde o usuário pode navegar entre as funcionalidades digitando o número correspondente.

Exemplo:

```
1 - Cadastro
2 - Cardápio
3 - Pedido
4 - Chat
5 - Ajuda
```
